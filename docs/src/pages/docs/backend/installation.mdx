## Manual

If you prefer to run the backend manually, you can follow these steps:

```bash
# Clone the repository
git clone git@github.com:krystxf/metro-now.git

# Open the backend directory
cd metro-now/backend

# Install dependencies
bun install

# Create a .env file
cp .env.example .env

# Edit the .env files with your own values

# Start the backend
bun start
```
