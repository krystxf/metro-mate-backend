# V1

## Metro departures

[Original data source](https://api.golemio.cz/pid/docs/openapi/#/%F0%9F%9A%8F%20PID%20Departure%20Boards/get_pid_departureboards)

Metro departures from platforms (GTFS ID)

### Endpoint

`/v1/metro/departures`

### Parameters

-   **Platform ID** (required)
    -   One platform (e.g. `gtfsID=U1040Z101P`)
    -   Multiple platforms (e.g. `gtfsID=U1040Z101P&gtfsID=U1040Z102P`)
