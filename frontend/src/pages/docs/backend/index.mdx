# Backend

[source code](https://github.com/krystxf/metro-now/tree/feat/documentation/backend)

- [Bun](https://github.com/oven-sh/bun)
- [WebSockets](https://developer.mozilla.org/en-US/docs/Web/API/WebSockets_API)
